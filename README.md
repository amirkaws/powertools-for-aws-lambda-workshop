## aws-lambda-powertools-typescript-workshop

## Deploy

* While in `backend` run `npm i` and then `cdk deploy`
* While in `frontend` run `npm i`, then `npm run build`, then `aws s3 sync 
build/ s3://${BUCKET_NAME}/`
